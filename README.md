# PerfDash (Interactive Personal Productivity Dashboard using Power BI)

## Project Objective
The PerfDash Productivity Dashboard is an interactive Power BI analytics dashboard designed to monitor and analyze personal productivity patterns. It tracks daily focus hours, tasks completed, distractions, sleep patterns, and goal achievement to provide meaningful insights into productivity trends and performance over time.

## Dataset used
<a href="https://github.com/kartikgautam1323/PerfDash/blob/main/Personal_Productivity_30Days.xlsx">Productivity Dataset</a>

## Question (KPIs)
- What is the total number of focus hours recorded during the tracking period?
- How many tasks were completed overall and per week?
- How many distractions occurred and how do they vary weekly?
- What percentage of goals were successfully achieved?
- How much sleep and break time was recorded across weeks?
- What is the overall productivity performance score based on tracked metrics?
  
- Dashboard Interaction <a href="https://github.com/kartikgautam1323/PerfDash/blob/main/Dashboard.png">View Dashboard</a>

## Process
- Imported productivity dataset into Power BI.
- Performed data cleaning and transformation using Power Query.
- Verified dataset for missing values, inconsistencies, and anomalies.
- Created data models and relationships where required.
- Developed DAX measures to calculate KPIs such as total focus hours, task completion, and performance score.
- Designed interactive visualizations including line charts, bar charts, donut charts, and KPI cards.
- Implemented filters and slicers to allow dynamic exploration of weekly and daily productivity trends.
